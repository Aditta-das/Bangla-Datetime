# Get Bangla Date 

## installation

pip install bd-datetime

## Usage

### YEAR

```
from bangladatetime import DateTime
d = DateTime()
print(d.year_())
``` 

### MONTH

```
from bangladatetime import DateTime
d = DateTime()
print(d.month_())
```

### DAY

```
from bangladatetime import DateTime
d = DateTime()
print(d.day_())
```
